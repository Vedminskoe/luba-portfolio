# Лендинг для услуг 3D-визуализатора

## Что это
Одностраничный сайт-визитка. Специалист — дизайнер интерьеров и ландшафтов.
Помогает презентовать проекты для потенциальных клиентов. Портфолио с выполненными проектами.

## Структура проекта
- index.html — единственный файл, всё внутри
- CLAUDE.md — этот файл

## Структура страницы (по блокам)
1. Навигация — Главная, Портфолио, Интерьер (цены и условия), Ландшафты (цены и условия), Контакты
2. Hero — полноэкранное фото работы, без текста поверх; ниже — название специальности и имя
3. «Обо мне» — три колонки: круглое фото → текст (опыт, образование, кол-во проектов, инструменты) → второе фото
4. Разделитель — декоративная отбивка без текста
5. Услуга №1 (интерьер) — фото + текст (что делаю → для кого → выгода) + 2 кнопки: «узнать цены и условия», «посмотреть портфолио»
6. Повтор навигации (дублируется в середине страницы)
7. Услуга №2 (ландшафты) — та же формула, зеркально
8. Финальный CTA — одна кнопка с обводкой «Заказать проект»
9. Футер — иконки WhatsApp/Telegram, копирайт-плейсхолдер, атрибуция конструктора

## Стиль
- Светлая тема, белый фон
- Монохромная база (чёрно-серый текст) + один акцентный цвет — приглушённый оливково-зелёный, только на кнопках
- Кнопки услуг — залитые оливковым, финальная CTA-кнопка — с обводкой, без заливки
- Кнопки — пилюли (полностью скруглённые), заглавные буквы, разреженный трекинг
- Шрифты: один семейный sans-serif на весь сайт — простой минималистичный для навигации и основного текста,
  тот же шрифт крупнее и с мягкими скруглёнными формами букв для заголовков блоков («обо мне», названия услуг)
- Много воздуха, низкая плотность контента, акцент на фотографиях-работах, а не на графике интерфейса

## Безопасность секретов

- Никогда не коммить .env, *.key, *.pem, credentials.json
  и любые файлы с реальными токенами и паролями
- Никогда не вставляй реальные значения секретов в код,
  README, комментарии, примеры
- В примерах вместо реальных значений используй заглушки:
  BOT_TOKEN=your-token-here
- Если видишь, что где-то в коде захардкожен секрет —
  сразу скажи мне, не делая коммита
- Все секреты читать через python-dotenv из .env
