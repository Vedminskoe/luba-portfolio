# Лендинг услуг техспеца

Одностраничный сайт-визитка. Специалист — Екамасова Любовь, дизайнер интерьеров и ландшафтный архитектор.

## Стек

- HTML, CSS, JavaScript — без сборщиков и фреймворков
- Весь сайт в одном файле `index.html`

## Как запустить

Открыть `index.html` в браузере. Никаких `npm install`, dev-серверов и сборки.
